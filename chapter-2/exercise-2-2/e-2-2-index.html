Exercise 2.2

Rewrite the applyForce() method using hte static method div() instead of get(). 

void applyForce(PVector force) {
  PVector f = __________._____(__________, __________);
  acceleration.add(f);
}

Solution

Even though I am using p5.js for all the exercises, to stay consistent with the exercise as presented, I will use Processing. 

void applyForce(PVector force) {
  PVector f = PVector.div(force, mass);
  acceleration.add(f);
}
