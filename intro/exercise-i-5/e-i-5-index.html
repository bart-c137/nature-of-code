Exercise I.5

A gaussian random walk is defined as one in which the step size (how far the object moves in a given direction) is generated with a normal distribution. Implement this variation of our random walk.

Solution

This is very similar to the Walker class the author describes in section I.2. There are 2 main changes

  • Drawing a line instead of drawing a point
  • Using the randomGaussian function instead of random.

In the Walker class, I added parameters to keep track of the previous x and y location so I can draw a line between the previous location and the new random location. 

class Walker {
  constructor() {
    this.x = width / 2;
    this.y = height / 2;

    this.previousX = this.x;
    this.previousY = this.y;

Now when a call to update is made, I first store the current x and y location in the previousX and previousY variables then get a random distance using the randomGaussian function. 

update() {
  this.previousX = this.x;
  this.previousY = this.y;

  let randomDistance = int(randomGaussian(this.mean, this.sd));
  this.randomMove(randomDistance);
}

I used 10 as the mean and 2 as the standard deviation, both are stored as variables in the class. The randomMove function simply determines which direction to move and the parameter passed in is the amount of pixels to move by, in this case the random number generated by the normal distribution. Drawing the line in the show function is drawn from the previous location to the new one. 

show() {
  stroke(0);
  line(this.previousX, this.previousY, this.x, this.y);
  ...
}

I also added metrics to confirm the distance values are from a normal distribution and that the direction of travel is from a uniform distribution.
