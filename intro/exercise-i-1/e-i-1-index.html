Exercise I.1

Create a random walker that has a tendency to move down and to the right.

Solution

Consider the walker should go right and down 30% of the time. In this case it should go left and up 20% of the time. A simple way to do this is

  1. Pick a random number from 1 through 10.
  2. If the value is between 1 and 3 inclusive, move right.
  3. If the value is between 4 and 6 inclusive, move down.
  4. If the value is between 7 and 8 inclusive, move left.
  5. If the value is between 9 and 10 inclusive, move up.

In code this is implemented as 

update() {
  let step = int(random(10));

  switch(step) {
  case 0:
  case 1:
  case 2:
      this.x++;
      break;
  case 3:
  case 4:
  case 5:
      this.y++;
      break;
  case 6:
  case 7:
      this.x--;
      break;
  case 8:
  case 9:
      this.y--;
      break;
  default:
      console.log("Unknonw random " + step);
  }
}

3 out of 10 is 30% and 2 out of 10 is 20%. Using the algorithm above, the walker will move down and to the right 30% of the time.
