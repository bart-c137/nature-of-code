Exercise I.9

Add a third argument to noise that increments once per cycle through draw() to animate the two- dimensional noise.

Solution

The canvas size is smaller because the large one caused problems on my PC.

Because the increment happens once per frame, the counter needs to be kept in the class itself, so I added a new variable, z, to the constructor. 

constructor() {
  this.x = 0;
  this.y = 0;

  this.z = 0.0;
}

Now it's as simple as adding this as a parameter to the noise call and incrementing it in the show function. 

// previous code before, see Exercise I.8

let c = noise(xoff, yoff, this.z) * 255;

// more code after

show() {
  this.z += 0.01;
}
