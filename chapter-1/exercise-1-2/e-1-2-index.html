Exercise 1.2

Take one of the walker examples from the introduction and convert it to use PVectors.

Solution

I decided to convert the basic random walker from Figure I.1 with 8 possible step directions. There are 2 basic changes to the original example

  • The location of the walker needs to be a PVector
  • The step taken by the walker must be a PVecotr

Converting the location to a PVector is easy enough 

constructor() {
  this.location = createVector(width / 2, height / 2);
}

Converting the step to a PVecotr is also not difficult. The random values picked in the example code can be used directly to create the vector, in my case named velocity. The random picking of the x and y values means the velocity vector is also random. Now velocity can be added to location. 

update() {
  let x = int(random(0, 3)) - 1;
  let y = int(random(0, 3)) - 1;

  let velocity = createVector(x, y);

  this.location.add(velocity);
}

Drawing the pint is just pulling the x and y values from the location vector. 

point(this.location.x, this.location.y);
