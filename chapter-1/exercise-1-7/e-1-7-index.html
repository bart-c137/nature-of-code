Exercise 1.7

Translate the following pseudocode to code using static or non-static functions where appropriate.

  • The PVector v equals (1, 5).
  • The PVector u equals v multiplied by 2.
  • The PVector w equals v minus u.
  • Divide the PVector w by 3.
PVector u = new PVector(1, 5);
PVector u = __________._______(_____,_____);
PVector w = __________._______(_____,_____);
____________;

Solution

Even though I am using p5.js for all the exercises, to stay consistent with the exercise as presented, I will use Processing. 

PVector u = new PVector(1, 5);
PVector u = PVector.mult(v, 2);
PVector w = PVector.sub(v, u);
w.div(3);
